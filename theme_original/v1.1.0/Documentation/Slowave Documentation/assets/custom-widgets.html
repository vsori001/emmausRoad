Custom Widgets in Slowave

We have a few custom widgets included in Slowave:

  1. Mailchimp Newsletter: This widget allows you to add a newsletter widget to your sidebars and footers. It requires an API key generated from your API account, after entering which you can assign the widget to single users up to specific mailchimp newsletters, ace!
  2. Ebor: Popular Posts: Is A widget which adds your most popular posts to your sidebar areas, ordered by comment count.
  3. Slowave Social Icons: Add some social profiles as a sidebar item using this widget.
