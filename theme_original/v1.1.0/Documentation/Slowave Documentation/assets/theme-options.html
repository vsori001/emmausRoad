Slowave Support

Theme Options

All of the theme options are controlled in "appearance" => "customise". You can also get to the same place with "Dashboard" => "slowave theme options"

Inside the theme options, take a look around, you have nearly full control of the entire theme in here :)

As you are changing colours and fonts, pay attention to the site, if something doesn't change, it's not because something is wrong, it's becuase there's likely greater control of that element within the page builder.

By combining the theme options and the page builder, you'll have an almost infinite ways of customising Slowave to your needs, without ever touching code.
