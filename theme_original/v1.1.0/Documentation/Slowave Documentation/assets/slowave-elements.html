Using slowave elements in the page builder

To use slowave elements in the page builder, first fire up the page builder on your page, then click the bottom left button "All Modules" to pull up the sections menu. all slowave menu elements are in sections marked "slowave"

  • Elements contains basic theme elements, image, video pricing table.
  • Tabs contains tabs, toggles, and the testimonials slider
  • Misc contains the section title, subtitle, seperator (horizontal line) etc.
  • Text contains slowave specific text blocks, the text blocks with icons, and the step by step section
  • Carousels holds the various carousels in the theme for various post types
  • Feeds contains the blog and portfolio feeds
