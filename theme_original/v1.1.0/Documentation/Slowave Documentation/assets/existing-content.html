Slowave Support

Existing Content

When you use the page builder it overwrites your existing page content (but saves it separately, so your old content is still safe!) that you have written in the WP Post Editor on the backend.

To use the existing content in the page builder, just use the page builder element "The Content", it will allow you to place your old page content right there into the page builder.

Alternatively, don't fire up the page builder on a page, and your exising content will show through instead, if you want to revert a page back to your existing content just delete all elements in the page builder for that page and update it, the content will revert back to your existing content.
