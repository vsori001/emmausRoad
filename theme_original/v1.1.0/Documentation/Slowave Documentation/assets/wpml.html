Slowave Support

WPML Support & Translating The Theme

Slowave is ready for WPML, the multilingual WordPress Plugin. To get started with the plugin, full details can be seen here:

WPML Documentation
Installing the plugin

WPML is a premium plugin, and so needs to be purchased, downloaded and then installed. You can buy WPML here:

http://wpml.org/purchase/

To install the plugin, grab the downloaded .zip and go to "plugins" => "add new" => "install" from your WP Dashboard.

Translating Theme Content

Slowave contains a number of words that are created by the theme, not by your content. To translate these it's best to use WPMLs string translation package. Full documentation of which is here:

String Translation

You can import the current .po file for the theme if you wish, you'll find slowave.po inside the /languages/ folder in the slowave theme files.

Translating your content within Slowave

Translating Content

Translating pages built by the page builder is incredibly easy:

  1. Build your page and add content in your native language.
  2. With the page builder open, click the "templates" tab (red) and use the "export" button, copy all the code that the page builder displays
  3. Go to the WP Edit screen for this page, go to the WPML seciton (around the publish settings) and duplicate the page into your next language.
  4. In the new page fire up the page builder, click the "templates" tab and use the "import" button. Paste the content we copied a few steps ago. Publish the page.
  5. All of your content for the page is now ready for you to translate into your next language.
