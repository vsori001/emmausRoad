Single Page Version Navigation

The navigation for the single page version of the theme is a little more complex, so read the following carefully.

Firstly, we'll be working with custom links in the menu rather than page items. See the following for details on making custom links:

Adding custom links to the menu

Here's the steps to make your single page navigation:

  1. First, build your single page, this is not covered in this section of the documetnation.
  2. For each Row you add to your single page, you're adding a "section" as the menu sees it, remember this, and plan your rows accordingly into the different menu sections you need.
  3. Go to "appearance" => "menus", make a new menu and assign it to the "single page navigation" location
  4. Now let's make some links, we're using the "links" option to do this. Each section you made on the page is numbered starting from 1. So to add a link to home on the single page we add the following as the link URL: #section-1
  5. Do this for the rest of your links, the second row is #section-2 etc. You can skip sections if you wish, just remember to count correctly the number of rows. If you skip a section make sure to continue your navigation in the same way, so if you add #section-1 #section-2 and then skip the third, you'd want to add #section-4
  6. That's it, you've made the menu :)

Other things to consider:

You need to go to "appearance" => "customise" => "site settings" and set "site version" to "single pager".

Only the single page template loads the single page navigaiton location in "appearance" => "menus", the rest of the site will still use the main navigation location, so you'll need another menu set in place in the "main navigation" location also. With the site set to "single pager" you can just add the same links to this menu, and the theme will redirect these back to your homepage (single page)
