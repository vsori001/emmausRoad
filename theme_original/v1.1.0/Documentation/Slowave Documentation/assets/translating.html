Translating the theme using the included .po files

  1. The .po file for the theme is in the /languages/ folder in your theme files.
  2. Open the included .po with POEdit (or similar), translate & create a .mo
  3. Name your .mo properly e.g pt_BR.mo
  4. Make a folder in the theme called languages & place your mo inside.
  5. Now add pt_BR to your WP-LANG settings in wp-config.php
