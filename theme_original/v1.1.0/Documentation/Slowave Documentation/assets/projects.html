Creating Project Posts

Add a New "Project Item; Adding a new Project item is just a matter of clicking "Add New" under the Project custom post type. To insert your image you just need to Upload an image and then set that image as your Featured Image.

Make sure that each Project item has a title, some Project-categories assigned, a featured image, and some content.

Project posts also have extra detail that you can complete, this detail is contained in the box 'Additional Project item details' underneath the WP Post Editor.

Your Project items will appear on your URL at /dslc_project/ by default, you can change this setting in the live composer plugin settings page.

Changing project post layouts

In the additional project details section, underneath the WP Editor, you can specify a layout to use, per project, as for what suits your content.

Adding broken up Projects to your Menu

If you want to divide your Projects into multiple, smaller Projects, you'll be adding Project categories to your menu. Project categories are visible at: http://www.your-site.com/dslc_projects_cats/the-category

Add this to your menu using the same method as above, you'll be given all the posts in that category, creating the ability to have multiple, smaller Projects

Changing the Project layout

To change the Project layout, head to the theme options at at Appearance => Customize.

Inside here you can modify your Project layout and change all the available options for the Project, any changes you make will be reflected at /Project

All Projects can be added into pages using the page builder!

You can add the projects feed to a page using the page builder, and specify categories, layout etc right there on the page!
