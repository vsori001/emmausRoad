Using Ebor Mega Menu

You'll notice in the demo of Slowave that we have a mega menu option, this is incredibly simple to use.

  1. Go to ebor mega menu post type on your WordPress Dashboard and make a new post
  2. Now go to "appearance" => "menus", look at the list of menu item types on the left of the menu screen, if Ebor Mega Menu is in there, great, select your new Ebor Mega Menu post and add it to your menu as a submenu item (Ebor mega menu items must be a submenu item, and they must be the only submenu item). If you don't see ebor mega menu, click on "screen options" in the top right of the screen, then in the dropdown tick "ebor mega menu" and repeat this step.
  3. With the post set as a submenu item in your menu, save and check your site, you should now have a wide dropdown bar where you added the item.
  4. Head back to your ebor mega menu post, we can now work with the post content to add whatever you need. You'll notice a dropdown above the editor "Insert Menu", which allows you to add menus into this post. So you can make extra menus in your menu editor, and use this add menu generator to add them to the mega menu.
  5. It's up to you what you add in here, but to make it a little more pretty, you can format your content with shortcodes. See the shortcodes section of this documentation for details on shortcodes included in Slowave.

That's it! Ebor mega menu should be seen then as a way to add post content directly into your menu, how you use it is entirely up to your own creativity!
